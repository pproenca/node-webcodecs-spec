# 7.10. Alpha Option

```webidl
enum AlphaOption {
  "keep",
  "discard",
};
```

Describes how the user agent _SHOULD_ behave when dealing with alpha channels, for a variety of different operations.

**keep**
: Indicates that the user agent _SHOULD_ preserve alpha channel data for [`VideoFrame`](#videoframe)s, if it is present.

**discard**
: Indicates that the user agent _SHOULD_ ignore or remove [`VideoFrame`](#videoframe)’s alpha channel data.
