# 7.3.1. AudioDecoderSupport

```webidl
dictionary AudioDecoderSupport {
  boolean supported;
  AudioDecoderConfig config;
};
```
**supported, of type boolean**
: A boolean indicating the whether the corresponding [`config`](#dom-audiodecodersupport-config) is supported by the User Agent.

**config, of type AudioDecoderConfig**
: An [`AudioDecoderConfig`](#dictdef-audiodecoderconfig) used by the User Agent in determining the value of [`supported`](#dom-audiodecodersupport-supported).
