# 7.3.2. VideoDecoderSupport

```webidl
dictionary VideoDecoderSupport {
  boolean supported;
  VideoDecoderConfig config;
};
```

**supported, of type boolean**
: A boolean indicating the whether the corresponding [`config`](#dom-videodecodersupport-config) is supported by the User Agent.

**config, of type VideoDecoderConfig**
: A [`VideoDecoderConfig`](#dictdef-videodecoderconfig) used by the User Agent in determining the value of [`supported`](#dom-videodecodersupport-supported).
