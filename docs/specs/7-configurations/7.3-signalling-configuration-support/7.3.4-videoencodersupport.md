# 7.3.4. VideoEncoderSupport

```webidl
dictionary VideoEncoderSupport {
  boolean supported;
  VideoEncoderConfig config;
};
```

**supported, of type boolean**
: A boolean indicating the whether the corresponding [`config`](#dom-videoencodersupport-config) is supported by the User Agent.

**config, of type VideoEncoderConfig**
: A [`VideoEncoderConfig`](#dictdef-videoencoderconfig) used by the User Agent in determining the value of [`supported`](#dom-videoencodersupport-supported).
