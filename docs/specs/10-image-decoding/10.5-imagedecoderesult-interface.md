# 10.5. ImageDecodeResult Interface

```webidl
dictionary ImageDecodeResult {
  required VideoFrame image;
  required boolean complete;
};
```
**image, of type VideoFrame**
: The decoded image.

**complete, of type boolean**
: Indicates whether [`image`](#dom-imagedecoderesult-image) contains the final full-detail output.

NOTE: [`complete`](#dom-imagedecoderesult-complete) is always `true` when [`decode()`](#dom-imagedecoder-decode) is invoked with [`completeFramesOnly`](#dom-imagedecodeoptions-completeframesonly) set to `true`.
