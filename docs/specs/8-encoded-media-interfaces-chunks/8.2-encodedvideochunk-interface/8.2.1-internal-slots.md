# 8.2.1. Internal Slots

**[[internal data]]**
: An array of bytes representing the encoded chunk data.

**[[type]]**
: The [`EncodedVideoChunkType`](#enumdef-encodedvideochunktype) of this [`EncodedVideoChunk`](#encodedvideochunk);

**[[timestamp]]**
: The presentation timestamp, given in microseconds.

**[[duration]]**
: The presentation duration, given in microseconds.

**[[byte length]]**
: The byte length of [`[[internal data]]`](#dom-encodedvideochunk-internal-data-slot).
