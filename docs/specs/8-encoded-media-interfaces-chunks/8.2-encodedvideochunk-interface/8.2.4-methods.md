# 8.2.4. Methods

**copyTo(destination)**
: When invoked, run these steps:

1.  If [`[[byte length]]`](#dom-encodedvideochunk-byte-length-slot) is greater than the [`[[byte length]]`](#dom-encodedvideochunk-byte-length-slot) of destination, throw a [`TypeError`](https://webidl.spec.whatwg.org/#exceptiondef-typeerror).
2.  Copy the [`[[internal data]]`](#dom-encodedvideochunk-internal-data-slot) into destination.
