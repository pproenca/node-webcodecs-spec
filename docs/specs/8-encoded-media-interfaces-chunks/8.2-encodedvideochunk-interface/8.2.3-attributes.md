# 8.2.3. Attributes

**type, of type EncodedVideoChunkType, readonly**
: Returns the value of [`[[type]]`](#dom-encodedvideochunk-type-slot).

**timestamp, of type long long, readonly**
: Returns the value of [`[[timestamp]]`](#dom-encodedvideochunk-timestamp-slot).

**duration, of type unsigned long long, readonly, nullable**
: Returns the value of [`[[duration]]`](#dom-encodedvideochunk-duration-slot).

**byteLength, of type unsigned long, readonly**
: Returns the value of [`[[byte length]]`](#dom-encodedvideochunk-byte-length-slot).
