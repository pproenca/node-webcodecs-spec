# 8.1.1. Internal Slots

**[[internal data]]**
: An array of bytes representing the encoded chunk data.

**[[type]]**
: Describes whether the chunk is a [key chunk](#key-chunk).

**[[timestamp]]**
: The presentation timestamp, given in microseconds.

**[[duration]]**
: The presentation duration, given in microseconds.

**[[byte length]]**
: The byte length of [`[[internal data]]`](#dom-encodedaudiochunk-internal-data-slot).
