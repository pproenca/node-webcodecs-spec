# 8.1.3. Attributes

**type, of type EncodedAudioChunkType, readonly**
: Returns the value of [`[[type]]`](#dom-encodedaudiochunk-type-slot).

**timestamp, of type long long, readonly**
: Returns the value of [`[[timestamp]]`](#dom-encodedaudiochunk-timestamp-slot).

**duration, of type unsigned long long, readonly, nullable**
: Returns the value of [`[[duration]]`](#dom-encodedaudiochunk-duration-slot).

**byteLength, of type unsigned long, readonly**
: Returns the value of [`[[byte length]]`](#dom-encodedaudiochunk-byte-length-slot).
