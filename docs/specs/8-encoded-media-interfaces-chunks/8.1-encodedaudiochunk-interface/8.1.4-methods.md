# 8.1.4. Methods

**copyTo(destination)**
: When invoked, run these steps:

1.  If the [`[[byte length]]`](#dom-encodedaudiochunk-byte-length-slot) of this [`EncodedAudioChunk`](#encodedaudiochunk) is greater than in destination, throw a [`TypeError`](https://webidl.spec.whatwg.org/#exceptiondef-typeerror).
    
2.  Copy the [`[[internal data]]`](#dom-encodedaudiochunk-internal-data-slot) into destination.
