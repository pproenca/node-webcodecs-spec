# 6.3. Attributes

**state, of type CodecState, readonly**
: Returns the value of [`[[state]]`](#dom-videoencoder-state-slot).

**encodeQueueSize, of type unsigned long, readonly**
: Returns the value of [`[[encodeQueueSize]]`](#dom-videoencoder-encodequeuesize-slot).

**ondequeue, of type EventHandler**
: An [event handler IDL attribute](https://html.spec.whatwg.org/multipage/webappapis.html#event-handler-idl-attributes) whose [event handler event type](https://html.spec.whatwg.org/multipage/webappapis.html#event-handler-event-type) is [`dequeue`](#eventdef-videoencoder-dequeue).
