# 4.3. Attributes

**state, of type CodecState, readonly**
: Returns the value of [`[[state]]`](#dom-videodecoder-state-slot).

**decodeQueueSize, of type unsigned long, readonly**
: Returns the value of [`[[decodeQueueSize]]`](#dom-videodecoder-decodequeuesize-slot).

**ondequeue, of type EventHandler**
: An [event handler IDL attribute](https://html.spec.whatwg.org/multipage/webappapis.html#event-handler-idl-attributes) whose [event handler event type](https://html.spec.whatwg.org/multipage/webappapis.html#event-handler-event-type) is [`dequeue`](#eventdef-videodecoder-dequeue).
