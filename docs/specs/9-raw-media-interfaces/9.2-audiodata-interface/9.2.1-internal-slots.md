# 9.2.1. Internal Slots

**[[resource reference]]**
: A reference to a [media resource](#media-resource) that stores the audio sample data for this [`AudioData`](#audiodata).

**[[format]]**
: The [`AudioSampleFormat`](#enumdef-audiosampleformat) used by this [`AudioData`](#audiodata). Will be `null` whenever the underlying format does not map to an [`AudioSampleFormat`](#enumdef-audiosampleformat) or when [`[[Detached]]`](https://html.spec.whatwg.org/multipage/structured-data.html#detached) is `true`.

**[[sample rate]]**
: The sample-rate, in Hz, for this [`AudioData`](#audiodata).

**[[number of frames]]**
: The number of [frames](#frame) for this [`AudioData`](#audiodata).

**[[number of channels]]**
: The number of audio channels for this [`AudioData`](#audiodata).

**[[timestamp]]**
: The presentation timestamp, in microseconds, for this [`AudioData`](#audiodata).
