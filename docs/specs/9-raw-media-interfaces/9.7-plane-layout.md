# 9.7. Plane Layout

[`PlaneLayout`](#dictdef-planelayout) [`VideoFrame`](#videoframe) [`BufferSource`](https://webidl.spec.whatwg.org/#BufferSource) [`PlaneLayout`](#dictdef-planelayout) _MAY_ [`VideoFrame`](#videoframe) [`copyTo()`](#dom-videoframe-copyto) [`BufferSource`](https://webidl.spec.whatwg.org/#BufferSource) [`copyTo()`](#dom-videoframe-copyto) [`PlaneLayout`](#dictdef-planelayout)
```webidl
dictionary PlaneLayout {
  [EnforceRange] required unsigned long offset;
  [EnforceRange] required unsigned long stride;
};
```
**offset, of type unsigned long**
: The offset in bytes where the given plane begins within a [`BufferSource`](https://webidl.spec.whatwg.org/#BufferSource).

**stride, of type unsigned long**
: The number of bytes, including padding, used by each row of the plane within a [`BufferSource`](https://webidl.spec.whatwg.org/#BufferSource).
